Data Cleaning Project

Overview
This project focuses on cleaning and preparing a raw sales dataset using Microsoft Excel.

The main goal of the project is to improve data quality by handling missing values, validating data consistency, and ensuring the dataset is ready for analysis.


Dataset Information
- Total Rows: 1200
- Total Columns: 14
- Dataset Type: Sales / Orders Data


Tasks Performed

1. Handling Missing Values
- Identified missing values in the CouponCode column
- Replaced blank values with "No Coupon"

2. Duplicate Check
- Verified that all OrderID values are unique
- No duplicate records were found

3. Data Formatting Validation
- Checked date formatting consistency
- Validated numerical columns
- Ensured text formatting consistency

4. Data Validation
- Verified that:
  
Quantity × UnitPrice = TotalPrice

- All records were accurate and consistent


Tools Used
- Microsoft Excel


Final Result
The dataset is now:
- Cleaned
- Consistent
- Ready for analysis
